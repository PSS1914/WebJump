[x] - Instale robotframework

[x] - Instale o Python 3.7 ou superior
[x] - Instale o Node.js 14.15 ou superior
[x] - Instalar o Git

    [x] Para linux ou mac: 
        ./run.sh

    [x] Para Windows: 
        ./run.bat
